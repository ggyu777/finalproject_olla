# Project : OllaEdu Sherpa

# 1. Propose
## 프로젝트에 대한 설명
- 데이터 기반 학생 종합 리포트 애플리케이션 개발 프로젝트
- pc, tablet, mobile에서 볼 수 있도록 반응형 작업
- context API로 전역상태관리

## 프로젝트 일정     
- 기획 : 2/11 ~ 2/17
- 1차 디자인 작업 : 2/18 ~ 2/22
